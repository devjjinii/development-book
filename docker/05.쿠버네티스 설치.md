## 쿠버네티스 설치

* Docker for mac/windows 를 설치했다면 별도의 설치과정이 필요없음.
* 개발 용도가 아닌, 실제 서비스 테스트 또는 운영 용도로 사용하려면 설치조건을 고려해야함.
  * AWS, GKE 등 클라우드 플랫폼, 온프레미스(자체 서버 환경)
